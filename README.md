## 배포 주소
https://fordogs-shop.vercel.app/

### 실행

- test
`npm run dev`

개발모드로 앱을 실행합니다.

http://localhost:5173

- 빌드
`npm run build` 후 `npm run preview`

배포 환경으로 실행합니다.

http://localhost:4173/

## 구현 기능
- Firebase Authentication을 사용하여 회원가입 / 로그인 페이지(구매자, 판매자 구분), 로그아웃
- 판매자 전용 페이지, 구매자 전용 페이지 분리
- 판매 상품 CRUD
- 장바구니 CRUD
- 판매 중인 상품 페이지
